# 🩺 Diabetes EDA — Pima Indians Dataset

## Overview
Exploratory Data Analysis on the Pima Indians Diabetes dataset
from UCI/Kaggle. Dataset contains 768 records of female patients
aged 21+ with 8 medical features.

## Tools Used

- Python, Pandas, NumPy
- Matplotlib, Seaborn

## Key Findings
- Glucose is the strongest predictor of diabetes
- 48.7% of Insulin values were missing (hidden as zeros)
- Dataset is imbalanced: 65% non-diabetic vs 35% diabetic
- Diabetes prevalence increases significantly after age 40
